ğŸ¯ MISSION 3.1: The Magic <img> Tag

ğŸ“– THE KNOWLEDGE: Image Elements

The <img> tag is SPECIAL - it's a self-closing tag (no closing </img> needed!) and it requires attributes to work.

ğŸ”‘ Required Attributes:

                                            
<img src="image.jpg" alt="Description">

src = "source" (path to image file)
alt = "alternative text" (describes the image)

âš ï¸ BOTH are mandatory!

  • src - tells browser WHERE the image is
  • alt - describes image for screen readers & if image fails

Image Format Best For Example
.jpg / .jpeg Photos, complex images photo.jpg
.png Logos, transparent backgrounds logo.png
.gif Animations, simple graphics animation.gif
.svg Icons, scalable graphics icon.svg

âœ¨ Examples:

                                            
<!-- Local image in same folder -->
<img src="cat.jpg" alt="Cute orange cat sleeping">

<!-- Image in subfolder -->
<img src="images/logo.png" alt="Company logo">

<!-- External image (full URL) -->
<img src="https://example.com/photo.jpg" alt="Mountain landscape">

âŒ COMMON MISTAKES:

                                            
WRONG: <img src="photo.jpg">           (missing alt!)
WRONG: <img alt="Photo">                (missing src!)
WRONG: <img src="photo.jpg" alt=""></img>  (don't use closing tag!)
CORRECT: <img src="photo.jpg" alt="Sunset at beach">

â™¿ Why Alt Text Matters

Alt text is crucial for:

  1. Screen readers - Visually impaired users hear the description
  2. Broken images - Shows when image fails to load
  3. SEO - Search engines can't "see" images, they read alt text
  4. Slow connections - Users know what's loading

Width and height are in PIXELS

Method Code When to Use
